# Creating Vite Project

## npm create vite@latest My-Project-Name -- --template react

// It will create react project

### Run Below Commands

- cd My-Project-Name
- npm install
- npm run dev

# CRA: - Create-React-App

## npx create-react-app My-Project-Name --template react

// It will create react project

### Run Below Commands

- cd My-Project-Name
- npm start
